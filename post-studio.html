Formatting Help

Write plain text. Use these only when you want formatting:

  • [b]bold[/b]
  • [i]italic[/i]
  • [code]code[/code]
  • [link https://example.com]link text[/link]
  • [center]centered text or link[/center]
  • [cta_image src="./assets/example.png" alt="Description" href="https://example.com"]PLAY THE GAME[/cta_image]
  • [image src="./assets/example.png" alt="Description"]Caption[/image]
  • [br] for a forced line break
  • Leave a blank line for a new paragraph
  • Start a line with - for bullet points
  • [table] ... [/table] for tables

Table example: [table][br]Name | League[br]Jeff "Brown Paper" Bagwell | MLB[br][/table]

Optional advanced tags: [text], [image], [embed], [pdf], [links].
